13.1 13장 1절 연습문제

연습문제 13.1 (SNU 13.1.1) 벡터장

\[ \boldsymbol{F}(x,\, y)= -2xy \boldsymbol{i} + (x^2-y^2)\boldsymbol{j} \]

를 도시하고, 이 벡터장은 중심이 \(x\) 축 위에 있고 원점을 지나는 임의의 원 \((x-a)^2+y^2 = a^2\) 에 접함을 보이라.

(해답).

\(f(x,\,y) = (x-a)^2+y^2-a^2\) 라 하자. \(f(x,\,y)=0\) 인 등위면을 생각하면, \[ \begin{aligned} \boldsymbol{F}(x,\,y) \cdot \nabla f(x,\,y) &= (-2xy,\, x^2-y^2) \cdot (2(x-a),\, 2y) \\ &= -2y(2x^2-2ax-x^2+y^2) \\ &= -2y(x^2-2ax+y^2) = 0 \end{aligned} \]

이다.


연습문제 13.2 (SNU 13.1.2) 벡터장 \(\boldsymbol{F}(x,\,y)= (-x,\,y)\) 는 \(xy=c\) 에 접함을 보이라.

(해답). \(f(x,\,y) = xy\) 라 하면,

\[ \boldsymbol{F}(x,\,y) \cdot \nabla f(x,\,y) = (-x,\,y) \cdot (y,\,x) = -xy+xy=0 \]


3. 생략


연습문제 13.3 (SNU 13.1.4) 위치벡터장 \(\boldsymbol{r}= x\boldsymbol{i} + y\boldsymbol{j} + z\boldsymbol{k}\) 에 대하여 벡터장

\[ \boldsymbol{F}(x,\,y,\,z) = \dfrac{-y\boldsymbol{i} + x\boldsymbol{j}}{r^3} = \boldsymbol{k} \times \dfrac{\boldsymbol{r}}{r^3} \]

의 크기가 일정한 점들의 집합을 도시하라.

(해답). \[ \begin{aligned} |\boldsymbol{F}(x,\,y,\,z)|=c & \implies \sqrt{x^2+y^2} =c(x^2+y^2+z^2)^{3/2} \qquad &; \rho = \sqrt{x^2+y^2}\\ & \rho = c(\rho^2+z^2)^{3/2} \end{aligned} \]

는 \(\dfrac{y}{(y^2+z^2)^{3/2}} = c\) 을 \(z\) 축 주위로 회전시킨 회전체면이다.


연습문제 13.4 (SNU 13.1.5 (각운동량 보존법칙)) 좌표공간에서 질량이 \(m\) 인 움직이는 점 \(X(t)\) 의 각운동량 (angular momentum) 은

\[ \boldsymbol{L}(t) :=X(t) \times (mX'(t)) \]

로 주어진다. 이 때 \(X(t)\) 가 뉴튼의 운동방정식 \(\boldsymbol{F}(X(t)) = m X''(t)\) 를 만족하고, 벡터장 \(\boldsymbol{F}\) 가 각 점에서의 위치벡터장에 비례하면 \(\boldsymbol{L}(t)\) 는 상수벡터임을 보이라.

(해답). \(\boldsymbol{F}(X(t)) = cX(t)\) 이므로,

\[ \dfrac{d}{dt}\boldsymbol{L}(t) = X'(t) \times (m X'(t)) + X(t) \times (m X''(t)) = 0 + cm X(t) \times X(t) = 0 \]


연습문제 13.5 (SNU 13.1.6) 주어진 벡터장 \(\boldsymbol{F}\) 에 대하여 곡선 \(X(t)\) 가 운동방정식 \(\boldsymbol{F}(X(t))=mX''(t)\) 를 만족시키면, 역향곡선 \(Y(s) :=X(c-s)\) 도 같은 운동방정식을 만족시킴을 보이라.

(해답). \[ \begin{aligned} Y'(s) &= -X'(c-s) \\ Y''(s) & = X''(c-s) \end{aligned} \]

이므로,

\[ \boldsymbol{F} (Y(t)) = \boldsymbol{F}(X(c-t) ) = mX''(c-t)=mY''(t) \]


13.2 13 장 2절 연습문제

연습문제 13.6 (SNU 13.2.1) 벡터장 \(\boldsymbol{F}(x,\,y) = (-y,\,x)\) 를 다음 곡선을 따라 적분하라.

(해답). (\(1\)) \(X(t) = (\cos t,\, \sin t),\qquad 0\le t \le 4\pi\)

\[ \begin{aligned} \int_X \boldsymbol{F}\cdot d\boldsymbol{s} & = \int_0^{4\pi} \boldsymbol{F}(X(t)) \cdot X'(t)\, dt = \int_0^{4\pi} (- \sin t,\, \cos t) \cdot (- \sin t,\, \cos t) \, dt \\ &= \int_0^{4\pi} 1 dt = 4\pi \end{aligned} \]

(\(2\)) \(X(t) = (\cos t,\, - \sin t), \qquad 0 \le t \le 2\pi\)

\[ \int_X \boldsymbol{F} \cdot d\boldsymbol{s} = \int_0^{2\pi} (\sin t,\, \cos t)\cdot (- \sin t,\, - \cos t)\, dt = -2\pi \]

(\(3\)) 생략


연습문제 13.7 (SNU 13.2.2) 벡터장

\[ \boldsymbol{a}(x,\,y,\,z) = \dfrac{-y\boldsymbol{i} + x \boldsymbol{j}}{x^2+y^2} \]

를 다음 곡선을 따라 점 \(P_0 = (1,\,0,\,0)\) 에서 \(P_1 = (0,\,1,\,1)\) 까지 적분하라.

(해답). (\(1\)) \(y=z=1-x\)

\(X(t) = (1-t,\,t,\,t),\, 0\le t \le 1\) 로 놓으면,

\[ \begin{aligned} \int_X \boldsymbol{a} \cdot d\boldsymbol{s} &= \int_0^1 \dfrac{(-t,\, 1-t,\,0)\cdot (-1,\,1,\,1)}{2t^2-2t+1}\, dt \\ &= \int_0^1 \dfrac{1}{2t^2-2t+1}\, dt = \int_0^1 \dfrac{1/2}{\left( t-\dfrac{1}{2}\right)^2 + \dfrac{1}{4}}\, dt\\ &=\int_{-1/2}^{1/2} \dfrac{1/2}{s^2+ 1/4}\, ds = \dfrac{\pi}{2} \end{aligned} \]


(\(2\)) \(X(t) = \left( \cos t,\, \sin t,\, \dfrac{2t}{\pi} \right),\qquad 0\le t \le \dfrac{\pi}{2}\)

\[ \begin{aligned} \int_X \boldsymbol{a} \cdot d\boldsymbol{s} & = \int_0^{\pi/2} \left( - \sin t,\, \cos t,\, 0\right) \cdot \left( - \sin t,\, \cos t,\, \dfrac{2}{\pi} \right)\, dt\\ &= \int_{0}^{\pi/2} 1\, dt = \dfrac{\pi}{2} \end{aligned} \]


(\(3\)) \(X(t) = \left(\cos t,\, -\sin t,\, \dfrac{2t}{3\pi} \right),\qquad 0 \le t \le \dfrac{3\pi}{2}\)

\[ \begin{aligned} \int_X \boldsymbol{a} \cdot d\boldsymbol{s} & = \int_0^{3\pi/2} \left(\sin t,\, \cos t,\,0 \right) \cdot \left(- \sin t,\, - \cos t,\, \dfrac{2}{3\pi}\right) \, dt \\ &= \int_0^{3\pi/2} -1 \, dt = - \dfrac{3\pi}{2} \end{aligned} \]


연습문제 13.8 (SNU 13.2.1 (감는수)) 좌표평면에서 원점을 지나지 않는 곡선

\[ X:[t_0,\,t_1] \to \mathbb{R}^2-\{O\} \]

에 대하여

\[ X(t) = |X(t)| (\cos \theta(t),\, \sin \theta (t)), \qquad t_0 \le t \le t_1 \]

이 되는 연속함수 \(\theta (t)\) 를 정할 수 있다. 이때

\[ \text{wind}(X) := \dfrac{1}{2\pi} \left(\theta(t_1) - \theta (t_0)\right) \]

를 곡선 \(X\) 가 원점을 감는 수(winding number) 라고 정의한다.

(\(1\)) 다음 곡선이 원점을 감는 수를 구하라.

 (\(a\)) \(X(t) := (\cos 3t,\, \sin 3t),\qquad 0 \le t \le 2\pi\),

 (\(b\)) \(Y(t) := (\cos t,\, \sin 3t),\qquad 0 \le t \le 2\pi\),

(\(2\)) 만약 \(X\) 가 폐곡선이면 감는 수가 정수(integer) 임을 설명하라.

(\(3\)) 만약 \(X\) 가 미분 가능한 곡선이면

\[ \text{wind}(X) = \dfrac{1}{2\pi} \int_X \boldsymbol{a}\cdot d\boldsymbol{s} \]

임을 보이라.

(해답). (\(a\)) \(X(t) := (\cos 3t,\, \sin 3t),\qquad 0 \le t \le 2\pi\)

극좌표계를 생각하면 \(\text{wind}(X)\) 는 반시계방향으로 원점 주위를 회전한 회전수임을 알 수 있다.

\(\theta (t) = 3t\) 이므로,

\[ \text{wind}(X) = \dfrac{1}{2\pi}(6\pi) = 3 \]


(\(b\)) \(Y(t) := (\cos t,\, \sin 3t),\qquad 0 \le t \le 2\pi\)

위의 그림에서 알 수 있듯이 반시계 방향으로 1번 회전하므로 \(-1\) 이다.

(\(2\)) 위에 했다.

(\(3\)) 텍스트에 나와 있음.

13.3 13장 3절 연습문제

연습문제 13.9 (SNU 13.3.1) 임의의 일급 함수 \(g(r)>0\) 에 대하여 벡터장

\[ \boldsymbol{F}(x,\,y) = g(r)(-y,\,x),\qquad (r=\sqrt{x^2+y^2}) \]

는 단위원을 한바퀴 돌며 적분한 값이 영이 아님을 보이고 이로부터 \(\boldsymbol{F}\) 는 잠재함수를 가지지 않는 다는 것을 설명하라.

(해답). \(\boldsymbol{F}(r,\,\theta) = rg(r)(-\sin \theta,\, \cos \theta)\) 이며, 단위원에 대해 적분하면,

\[ \begin{aligned} \int_C \boldsymbol{F}(x,\,y)\, d\boldsymbol{s} &= \int_0^{2\pi} g(1) (- \sin \theta,\, \cos \theta)\cdot (-\sin \theta,\, \cos \theta)\, dt\theta\\ &= \int_0^{2\pi} g(1) d\theta = 2\pi g(1) > 0 \end{aligned} \]


연습문제 13.10 (SNU 13.3.2) 벡터장

\[ \boldsymbol{F}(x,\,y,\,z) = yz\boldsymbol{i} + (xz+y)\boldsymbol{j} + (xy+z^2)\boldsymbol{k} \]

의 잠재함수를 구하라.

(해답). \(\phi (x,\,y,\,z)\) 를 잠재함수라 하면,

\[ \partial_x \phi = yz,\, \partial_y \phi = xz+y,\, \partial_z \phi = xy+z^2 \]

이다. 첫번째 편매분으로부터 \(\phi = xyz + g(y,\, z)\) 임을 안다. 두번째 편미분으로부터 \(\partial_y g(y,\,z) = y\) 임을 알수 있으므로 \(g(y,\,z) = \dfrac{1}{2}y^2+ h(z)\) 임을 안다. 따라서, \(\phi = xyz + \dfrac{1}{2}y^2 + h(z)\) 이고, 세번째 편미분으로부터

\[ \phi (x,\,y,\,z) = xyz + \dfrac{1}{2}y^2 + \dfrac{1}{2}z^2 + \text{const.} \]

임을 안다.


연습문제 13.11 (SNU 13.3.3) 벡터장 \(\boldsymbol{F}(x,\,y) = -(y,\,x)\) 와 곡선

\[ X(t) := (\cos t,\, \sin t),\qquad 0 \le t \le \pi \]

에 대하여 선적분 \(\int_X \boldsymbol{F}\cdot d\boldsymbol{s}\) 의 값을 구하라. 또,

\[ \tilde{X}(t) := (\cos 3t,\, \sin 3t),\qquad 0 \le t \le \pi \]

로 두면, 두 곡선

\[ \tilde{X},\, X : [0,\, \pi] \to \mathbb{R}^2 \]

은 서로 같은 경례글 가지지만

\[ \int_{\tilde{X}} \boldsymbol{F}\cdot d\boldsymbol{s} \ne \int_X \boldsymbol{F}\cdot d\boldsymbol{s} \]

임을 보여라.

(해답). \[ \begin{aligned} \int_X \boldsymbol{F}\cdot d\boldsymbol{s} &= \int_0^{\pi}( -\sin t,\, \cos t) \cdot (- \sin t,\, \cos t)\, dt = \pi \\ \int_{\tilde{X}}\boldsymbol{F} \cdot d\boldsymbol{s} &= \int_0^\pi (- \sin 3t,\, \cos 3t) \cdot (-3 \sin 3t,\, 3 \cos 3t )\, dt = 3\pi \end{aligned} \]


연습문제 13.12 (SNU 13.3.4) 벡터장 \[ \boldsymbol{F}(x,\,y) = (x^2+y^2)\boldsymbol{i} + 2xy \boldsymbol{j} \]

의 잠재함수를 구하고, 벡터장의 모양을 그리라.

(해답). \(\phi (x,\,y) = \dfrac{1}{3}x^3 + xy^2+ \text{const.}\)


연습문제 13.13 (SNU 13.3.5) 벡터장 \(\boldsymbol{F}(x,\,y,\,z) = (0,\, -z,\,y)\) 는 잠재함수를 가지지 않음을 보여라.

(해답). \(D_3 f_2 = -1 \ne 1 = D_2 f_3\)


연습문제 13.14 (SNU 13.3.6) \(n\) 공간에서 다음 벡터장의 잠재함수를 구하라.

 (\(1\)) \(\boldsymbol{F}(X) = X/|X|\),

 (\(2\)) \(\boldsymbol{F}(X) = X/|X|^3\),

 (\(3\)) \(\boldsymbol{F}(X) = |X|^k X \qquad (k \ne -2)\).

(해답). (\(1\))

\[ \phi (X) = |X| \]

(\(2\)) \[ \phi (X) = -1/|X| \]

(\(3\))

\[ \phi (X) = \dfrac{1}{k+2}|X|^{(k+2)} \]


연습문제 13.15 (SNU 13.3.7) 다음 벡터장의 잠재함수를 구하라

 (\(1\)) \(\boldsymbol{F}(x,\,y,\,z) = (yz,\,zx,\,xy)\),

 (\(2\)) \(\boldsymbol{F}(x,\,y,\,z) = (2x-y-z,\, 2y-x,\,2z-x)\),

 (\(3\)) \(\boldsymbol{F}(x,\,y,\,z) = (y \cos z - yze^x,\, x\cos z - ze^x,\, -xy\sin z - ye^x)\),

\[ \phi (x,\,y,\,z) = xyz+\text{const.} \]

(\(2\))

(\(3\))

\[ \phi (x,\,y,\,z) = xy \cos z - yz e^x + \text{const.} \]

(\(4\))

\[ \phi (x,\,y,\,z) = xyz+ \dfrac{1}{2}y^2 + z + \text{const.} \]

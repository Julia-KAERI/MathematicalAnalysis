15.1 다변수 함수의 적분

15.1.1 정의와 기본적 성질

  • \(\mathbb{R}\) 에서의 분할은 정의 10.1 을 참고하라. 전체적으로 리만-스틸체스 적분 를 참고하라.

  • 여기서의 \(n\)-cell 은 특별한 언급이 없는 한 닫힌 \(n\)-cell 을 의미한다.

정의 15.1 (\(n\)-cell 의 부피) \(\mathbb{R}^n\) 에서 정의된 \(n\)-cell \(Q=[a_1,b_1] \times \cdots \times [a_n,b_n]\) 의 부피 \(\text{vol(Q)}\) 는 다음과 같이 정의된다.

\[ \text{vol}(Q) := (b_1-a_1) (b_2-a_2)\cdots (b_n-a_n). \]

정의 15.2 (\(n\)-cell 의 분할과 subcell) 닫힌 \(n\)-cell \(Q=[a_1,\,b_1] \times \cdots \times [a_n,\,b_n]\) 을 생각하자. 각각의 \(i=1,\ldots,\,n\) 에 대해 \(P_i\) 가 \([a_i,\,b_i]\) 의 분할 일 때 \(P = (P_1,\ldots,\,P_n)\) 를 \(Q\) 의 분할 (partition) 이라고 한다. \(Q\) 에 대한 모든 분할의 집합을 \(\mathcal{P}[Q]\) 라고 표기한다.

\(n\) cell \(q=[a^{(1)}_{i_1},\,a^{(1)}_{i_1+1}] \times \cdots \times [a^{(n)}_{i_n},\,a^{(n)}_{i_{n}+1}]\) 가 각각의 \(k=1,\ldots,\,n\) 에 대해 \(a^{(k)}_{i_k},\, a^{(k)}_{i_k+1} \in P_k\) 일 때 이 \(n\)-cell \(q\) 를 분할 \(P\) 에 의한 \(Q\) 의 subcell 이라고 한다.


짐작하듯이 \(n\)-cell \(Q\) 에 대한 분할 \(P\) 가 정해지면 \(n\)-cell 에 대한 모든 subcell 도 정해진다.


정의 15.3 (refinement) \(n\)-cell \(Q\) 에 대한 두 분할 \(P_1\), \(P_2\) 를 생각하자. \(P_1\) 에 의한 subcell 이 모두 \(P_2\) 에 의한 subcell 이거나 \(P_2\) 의 subcell 에 포함된다면 \(P_2\) 를 \(P_1\) 의 refinement 라고 한다. \(P_2\) 가 \(P_1\) 의 refinement 일 때 \(P_1 \subset P_2\) 라고 표기한다.


정의 15.4 (리만 상합, 리만 하합) \(Q\subset \mathbb{R}^n\) 이 \(n\)-cell 이고 \(Q\) 에 대해 정의된 함수 \(f:Q \to \mathbb{R}\) 이 유계라고 하자. \(P\) 가 \(Q\) 의 분할이며 \(S\) 가 \(P\) 의 subcell 일 때

\[ \begin{aligned} M_S(f) &:= \sup \{f(\boldsymbol{x}) : \boldsymbol{x}\in S\}, \\[0.3em] m_S(f) &:= \inf \{f(\boldsymbol{f}): \boldsymbol{x}\in S\} \end{aligned} \]

라고 하자. \(\text{vol}(S)\) 를 cubcell \(S\) 의 부피라고 할 때 분할 \(P\) 에 대한 \(f\) 의 리만 상합 \(U(f,\, P)\) 와 리만 하합 \(L(f,\,P)\) 는 다음과 같이 정의된다.

\[ \begin{aligned} U(f,\,P) := \sum_{S} m_S \cdot \text{vol}(S), \\[0.3em] L(f,\,P) := \sum_{S} M_S \cdot \text{vol}(S). \end{aligned} \]


명제 15.1 \(n\)-cell \(Q\) 에서 정의된 유계인 함수 \(f : Q \to \mathbb{R}\) 와 \(P,\,P_1,\,P_2\in \mathcal{P}[Q]\) 에 대해 다음이 성립한다.

 (\(1\)) \(L(f,\,P) \le U(f,\, P)\).

 (\(2\)) \(P_1 \subset P_2 \implies L(f,\,P_1) \le L(f,\, P_2),\, U(f,\,P_2) \le U(f,\,P_1)\).

 (\(3\)) \(L(f,\, P_1) \le U(f,\,P_2)\)

(증명). (\(1\)) 각각의 subcell \(S\) 에 대해 \(m_S \le M_S\) 이므로 자명하다.

(\(2\)) \(S\) 가 \(P_1\) 에 의한 subcell 일 때 하나 이상의 \(P_2\) 의 subcell \(S_1,\ldots,\,S_m\) 의 합집합이다. 이 때 각각의 \(i=1,\ldots,\,m\) 에 대해 \(m_S \le m_{S_i}\) 이며 \(M_S \ge M_{S_i}\) 이고 \(\text{vol}(S) = \sum_{i=1}^m \text{vol}(S_i)\) 이므로 (\(2\)) 가 성립한다.

(\(3\)) \(P_1\) 과 \(P_2\) 의 공동의 refinement \(P_3\) 를 생각 할 수 있다. 그렇다면 (\(2\)) 에 의해

\[ L(f,\,P_1) \le L(f,\, P_3) \le U(f,\, P_3) \le L(f,\,P_2) \]

이다. \(\square\)


명제 15.1 의 (\(3\)) 으로부터 \(f:Q\to \mathbb{R}\) 에 대해 \(\{L(f,\,P): P\in \mathcal{P}[Q]\}\) 와 \(\{U(f,\,P) : P\in \mathcal{P}[Q]\}\) 는 유계임을 안다. 또한

\[ \sup \{L(f,\,P):P\in \mathcal{P}[Q]\} \le \inf \{ U(f,\,P): P\in \mathcal{P}[Q] \} \]

임도 안다. 일변수 함수의 적분에서와 같이 다변수 실함수에 대한 리만 적분을 정의 할 수 있다.

정의 15.5 (리만 적분) \(n\)-cell \(Q\) 에서 정의된 유계인 함수 \(f:Q\to \mathbb{R}\) 에 대해 리만 상적분 \(\displaystyle \overline{\int_Q}f\) 와 리만 하적분 \(\displaystyle \underline{\int_Q}f\) 를 다음과 같이 정의한다.

\[ \underline{\int_Q} f := \sup \{L(f,\,P):P\in \mathcal{P}[Q]\},\qquad \overline{\int_Q} f := \inf \{ U(f,\,P): P\in \mathcal{P}[Q] \} \]

\(\displaystyle \underline{\int_Q} f = \displaystyle \overline{\int_Q} f\) 일 때 \(f\) 가 \(Q\) 에서 리만 적분 가능하다(Riemann-integrable), 혹은 단순히 적분 가능하다(integrable) 고 하고 이 값을

\[ \int_Q f \]

라고 쓴다. \(Q\) 에서 적분 가능한 모든 함수의 집합을 \(\mathcal{R}[Q]\) 라고 쓴다.


일변수 함수에서의 정리 10.6 와 유사한 아래 정리는 아주 유용하다.


정리 15.1 \(n\)-cell \(Q\) 에서 정의된 유계함수 \(f:Q \to \mathbb{R}\) 에 대해 다음은 동치이다.

 (\(1\)) \(f\in \mathcal{R}[Q]\),

 (\(2\)) 임의의 \(\varepsilon >0\) 에 대해 \(U(f,\,P)-L(f,\,P)<\varepsilon\) 인 \(P\in \mathcal{P}[Q]\) 가 존재한다.

(증명). (\(1\implies 2\)) \(\sup \{L(f,\, P)\} = \int_Q f\) 이므로 \(\int_Q f - L(f,\,P_1)<\varepsilon/2\) 를 민족하는 \(P_1\in \mathcal{P}[Q]\) 가 존재한다. 마찬가지로 \(U(f,\,P_2) - \int_Q f < \varepsilon/2\) 를 만족하는 \(P_2\in \mathcal{P}[Q]\) 가 존재한다. \(P_1,\,P_2\) 의 공통 refinement \(P\) 에 대해서도 두 부등식이 성립하며 두 부등식을 더하면 \(U(f,\,P) - L(f,\,P) \le \varepsilon\) 이다.

(\(2\implies 1\)) \(\displaystyle L(f,\,P) \le \underline{\int_Q}f \le \overline{\int_Q} f \le U(f,\,P)\) 이므로 임의의 \(\varepsilon>0\) 에 대해

\[ \overline{\int_Q} f - \underline{\int_Q}f < \varepsilon \]

이다. 이는 상적분과 하적분 값이 같음을 의미하므로 \(f\) 는 리만 적분 가능하다. \(\square\)


연습문제

연습문제 15.1 (Spivak 3.1) \(f: [0,\,1]\times [0,\,1] \to \mathbb{R}\) 이 다음과 같이 정의되었다.

\[ f(x,\,y) = \left\{ \begin{array}{ll} 0, \qquad & \text{if } 0 \le x < \frac{1}{2}, \\ 1 & \text{if } \frac{1}{2} \le x < 1. \end{array}\right. . \]

\(f\) 가 적분가능하며 \(\int_{[0,\,1]\times [0,\,1]} f = \frac{1}{2}\) 임을 보여라.

(해답). 임의의 \(0<\varepsilon <1\) 에 대해 \([0,1]\times [0,\,1]\) 에 대한 분할 \(P\) 를 다음과 같이 잡는다.

\[ P = \left\{ \left[0,\, \dfrac{1}{2}-\dfrac{\varepsilon}{4}\right] \times [0,\,1], \, \left[\dfrac{1}{2}-\dfrac{\varepsilon}{4},\, \dfrac{1}{2}+\dfrac{\varepsilon}{4}\right] \times [0,\,1], \, \left[\dfrac{1}{2}+\dfrac{\varepsilon}{4},\, 1\right] \times [0,\,1] \right\} \]

이 때

\[ U(f,\,P) = \dfrac{2+\varepsilon}{4},\qquad L(f,\,P) = \dfrac{2-\varepsilon}{4} \]

이므로

\[ U(f,\,P)-L(f,\,P) = \dfrac{\varepsilon}{2} < \varepsilon \]

이다. 정리 15.1 에 의해 \(f\) 는 적분가능하다.


연습문제 15.2 (Spivak 3.2) \(f:A \to \mathbb{R}\) 이 적분가능하며 \(g\) 는 유한개의 점을 제외하고는 \(f\) 와 같다. \(g\) 가 적분가능하며 \(\int_A f = \int_A g\) 임을 보여라.

(해답). \(B=\{\boldsymbol{x}\in A : f(\boldsymbol{x})\ne g(\boldsymbol{x})\}\) 라고 하고 \(M=\max \{|f(\boldsymbol{x})-g(\boldsymbol{x})| : \boldsymbol{x}\in B\}\) 이라고 하자. \(N_B=|B|\), 즉 \(B\) 의 개수 라고 하자.

\(P\in \mathcal{P}[A]\) 를 (\(i\)) \(\boldsymbol{x}\in B\) 가 각각 \(P\) 의 서로 다른 subcell 내부에 위치하며, (\(ii\)) \(P\) 의 각각의 subcell 의 부피가 \(V_0=\varepsilon/4N_BM\) 보다 작고 (\(iii\)) \(U(f,\,P)- L(f,\,P)<\varepsilon/2\) 가 되도록 잡을 수 있다. 그렇다면

\[ \begin{aligned} U(f,\,P) - N_BV_0M &\le U(g,\,P) \le U(f,\,P) + N_BV_0M \\[0.3em] L(f,\,P) - N_BV_0M & \le L(g,\,P) \le L(f,\,P) + N_BV_0M \end{aligned} \tag{15.1}\]

이므로

\[ U(g,\,P) - L(g,\,P) \le U(f,\,P)-L(f,\,P) + 2N_BV_0M < \varepsilon \]

이므로 \(g\in \mathcal{R}[A]\) 이다. 식 15.1 로부터

\[ L(f,\,P) - \dfrac{\varepsilon}{2} \le \int_A g \le U(f,\,P) + \dfrac{\varepsilon}{2} < L(f,\,P) + \varepsilon \]

이다. 임의의 \(\varepsilon>0\) 에 대해 성립하므로 \(\int_A g = \int_A f\) 임을 알 수 있다.


연습문제 15.3 (Spivak 3.3) \(f,\,g:A \to \mathbb{R}\) 은 적분가능하다.

(\(a\)) 임의의 분할 \(P\) 와 \(P\) 에 의한 \(n\)-cell \(S\) 에 대해

\[ m_S(f)+m_S(g) \le m_S(f+g),\qquad M_S(f+g) \le M_S(f)+M_S(g) \tag{15.2}\]

이며, 따라서

\[ \begin{eqnarray} L(f,\,P) + L(g,\,P) \le L(f+g,\,P) \le U(f+g,\,P) \le U(f,\,P) + U(g,\,P) \end{eqnarray} \tag{15.3}\]

임을 보여라.

(\(b\)) \(f+g\) 는 적분가능하며 \(\int_A(f+g) = \int_A f+ \int_Ag\) 임을 보여라.

(\(c\)) 어떤 상수 \(c\) 에 대해 \(\int_A(cf)= c\int_Af\) 임을 보여라.

(해답). (\(a\)) trivial

(\(b\)) 임의의 \(\varepsilon>0\) 에 대해 \(U(f,\,P_1)-L(f,\,P_1)<\frac{\varepsilon}{2}\) 인 \(A\) 의 분할 \(P_1\) 과 \(U(g,\,P_2)-L(g,\,P_2)<\frac{\varepsilon}{2}\) 인 \(A\) 의 분할 \(P_2\) 가 존재한다. \(P_1\) 과 \(P_2\) 의 공통 refinement 인 \(P\) 에 대해서도 위의 부등식이 성립하며,

\[ U(f+g,\,P)-L(f+g,\,P) \le U(f,\,P)-L(f,\,P) + U(g,\,P)-L(g,\,P) < \varepsilon \]

이다. 따라서 \(f+g \in \mathcal{R}[A]\) 이다. 임의의 \(\varepsilon>0\) 에 대해 \(\int_A f - L(f,\,P_1)<\frac{\varepsilon}{2}\), \(\int_A g - L(g,\,P_1)<\frac{\varepsilon}{2}\) 인 분할 \(P_1\) 이 존재하며 \(U(f,\,P_2)-\int_A f<\frac{\varepsilon}{2}\), \(U(g,\,P_2)-\int_A g<\frac{\varepsilon}{2}\) 인 분할 \(P_2\) 가 존재한다. \(P\) 를 \(P_1\) 과 \(P_2\) 의 공통의 refinement 라고 하면 위의 부등식이 모두 성립하며

\[ \left(\int_A f + \int_A g \right) -\varepsilon \le \int_A (f+g) \le \left(\int_A f + \int_A g \right)+ \varepsilon \]

이다. 임의의 \(\varepsilon>0\) 에 대해 성립하므로 \(\displaystyle \int_A (f+g) = \int_A f + \int_A g\) 이다.

(\(c\)) trivial.


연습문제 15.4 (Spivak 3.4) \(f:A \to \mathbb{R}\) 과 \(P\in \mathcal{P}[A]\) 를 생각하자. \(f\in \mathcal{R}[A]\) 일 필요충분조건은 각각의 \(P\) 의 \(n\)-cell \(S\) 에 대해 \(f|_S \in \mathcal{R}[S]\) 인 것임을 보여라. 그리고 이 경우 \(\int_A f = \sum_S \int_S f|_S\) 임을 보여라.

(해답). \(A\) 에 대한 분할 \(P\) 에 의해 정해진 \(n\)-cell 의 집합을 \(\mathcal{S}[P]\) 라고 하자. 이 때 다음은 자명하다.

\[ L(f,\,P) = \sum_{S \in \mathcal{S}[P]} L(f|_S,\, S) ,\qquad U(f,\,P) = \sum_{S \in \mathcal{S}[P]} U(f|_S,\, S) \tag{15.4}\]

여기서 \(U(f|_S,\,S)-L(f|_S,\,S) \le U(f,\,P)-L(f,\,P)\) 이며 임의의 정해진 \(\varepsilon<0\) 에 대해 \(U(f,\,P)-L(f,\,P)<\varepsilon\) 인 분할 \(P\) 가 존재하며 이 분할에 대해 \(U(f|_S,\,S)-L(f|_S,\,S)<\varepsilon\) 이므로 \(f|_S\) 는 \(S\) 에서 적분가능하다.

이제 각각의 \(f|_S\) 가 적분가능함을 가정하자. \(P\) 에 의한 \(n\)-cell 이 \(N\) 개라고 하자. \(S\) 에 대한 분할 \(P_S\) 에 대해

\[ U(f|S,\, P_S)-L(f|S,\, P_S) \le \dfrac{\varepsilon}{N} \]

이 성립하며, 각각의 \(P_S\) 에 대한 공통의 refinement \(P'\) 에 대해

\[ U(f,\,P')-L(f,\,P') \le \varepsilon \]

이므로 \(f\) 는 적분 가능하다. \(\int_A = \sum_S \int_S f|_S\) 임은 식 15.4 로 부터 자명하다.


연습문제 15.5 (Spivak 3.5) \(f,\,g: A \to \mathbb{R}\) 가 적분가능하고 \(f\le g\) 라면 \(\int_A f \le \int_A g\) 임을 보여라.

(해답). \(h=g-f\) 이면 \(h\ge 0\) 이다. 임의의 분할 \(P\) 에 대해 \(U(f,\,P) \ge L(f,\,P) \ge 0\) 이므로 \(\int_A h \ge 0\) 이다. 연습문제 15.3 에 의해

\[ \int_A g- \int_A f = \int_A (g-f) = \int_A h \ge 0 \]

이다.


연습문제 15.6 (Spivak 3.6) \(f:A \to \mathbb{R}\) 이 적분가능하다면 \(|f|\) 역시 적분가능하며 \(\left|\int_A f\right| \le \int_A|f|\) 임을 보여라.

(증명). \(f^+(x) := \max (f(x),\,0),\, f^-(x) = \min (f(x),\,0)\) 이라고 하자. \(f = f^+ + f^-\) 이며 \(|f|=f^+ - f^-\) 이다. \(A\) 에 포함되는 임의의 \(n\)-cell \(S\) 에 대해

(\(i\)) \(M_s(f)\ge 0,\, m_s(f) \ge 0\) 이면 \(M_S(f^+) = M_S(f)\), \(m_s(f^+) = m_S(f)\), \(M_S(f^-)=M_S(f^-)=0\)

(\(ii\)) \(M_s(f) \ge0,\, m_S(f) \le 0\) 이면 \(M_S(f^+) = M_S(f),\, m_S(f^{+})=0\), \(M_S(f^-)=0\), \(m_S(f^-) = m_S(F)\),

(\(iii\)) \(M_s(f) \le 0,\, m_S(f) \le 0\) 이면 \(M_S(f^+) = m_S(f^{+})=0\), \(M_S(f^-)=M_S(f)\), \(m_S(f^-) = m_S(F)\),

이다. \(M_S(f^+) - m_S(f^+) \le M_S(f)- m_S(f)\) 이며 \(M_S(f^-)-m_S(f^-)\le M_S(f)- m_S(f)\) 이므로

\(f^+\) 와 \(f^-\) 는 적분가능하며 \(|f| = f^+ - f^-\) 이므로 \(|f|\) 도 적분가능하다. 또한

\[ \left|\int_A f \right| = \left|\int_A (f^+ + f^-)\right| \le \left|\int_A f^+ \right| + \left|\int_a f^-\right| = \int_A |f| \]

이다.


연습문제 15.7 (Spivak 3.7) \(f:[0,\,1]\times [0,\,1]\to \mathbb{R}\) 이 다음과 같이 정의되었다.

\[ f(x,\,y) = \left\{\begin{array}{ll} 0, \qquad & x\in \mathbb{R}\backslash \mathbb{Q}, \\ 0, & x\in \mathbb{Q},\, y \in \mathbb{R}\backslash\mathbb{Q}\\ 1/q & x\in \mathbb{Q},\, y=p/q \text{ in lowest terms}. \end{array}\right. \]

\(f\) 가 적분가능하며 \(\int_{[0,\,1]\times [0,\,1]} f = 0\) 임을 보여라.

(해답). 임의의 partition \(P\in \mathcal{P}{[0,1]\times [0,\,1]}\) 에 대해 \(L(f,\,P)=0\) 임은 자명하다.

\(f(x,\,y)=1\) 인 값은 \(y=1\) 인 경우밖에 없다. \(f(x,\,y)=1/2\) 인 것은 \(y=2\) 인 경우, \(f(x,\,y)=1/3\) 인 것은 \(y=1/3,\, 2/3\) 인 경우이다. 즉 \(f(x,\,y)=1/q\) 인 \(y\) 값은 \(q=1\) 인 경우를 제외하면 최대 \(q-1\) 개이다.

\([0,\,1] \times [a_i,\,b_i]\) 형태의 partition 을 생각한다. 이제 subcell \(S_1 = [0,\,1] \times [0,\, 1/2]\) 로 잡고 \(q\in \mathbb{Z}_+, \, q\ge 2\) 에 대해 \(S_q\) 를 \(f(x,\,y)=1/q\) 가 가능한 \(y\) 값을 중심으로 \(x\) 축방향으로 폭이 1, \(y\) 축 방향으로 폭이 \(1/2^{q}\) 만큼 되도록 잡자. 겹치는 부분이 있어도 \(U(f,\,P)\) 의 상한을 정하는 것이므로 상관 없이 계산한다. \(f(x,\,y) = 1/q\) 인 값은 최대 \(q-1\) 개임을 다시 기억한다. \(q=1,\ldots,\, m\) 까지 계산하면 여기에 포함되지 않은 면적은 \(\dfrac{1}{2} + \cdots + \dfrac{1}{2^{m+1}}\) 만큼이며 여기에서의 함수값은 \(\dfrac{1}{m+1}\) 보다 작으므로

\[ \begin{aligned} U(f,\,P) &\le 1 \cdot \dfrac{1}{2} + \dfrac{1}{2}\cdot \dfrac{1}{2^2}\cdot (2-1) + \dfrac{1}{3} \cdot \dfrac{1}{2^3}\cdot (3-1)+\cdots \dfrac{1}{m}\cdot \dfrac{1}{2^m}\cdot (m-1) \\ &\quad + \dfrac{1}{m+1} \left[1- \left(\dfrac{1}{2} + \dfrac{1}{2^2} + \cdots + \dfrac{1}{2^m} \right)\right] \\ &\le \dfrac{1}{2} + \dfrac{1}{2^2} + \cdots + \dfrac{1}{2^m} + \dfrac{1}{m+1} \left[1- \left(\dfrac{1}{2} + \dfrac{1}{2^2} + \cdots + \dfrac{1}{2^m} \right)\right] \\ &=\dfrac{1}{m+1} \end{aligned} \]

이다. 정해진 작은 \(\varepsilon>0\) 에 대해 \(m>\dfrac{1}{\varepsilon} -1\) 로 잡으면 \(L(f,\,P)=0\) 이므로

\[ U(f,\,P) - L(f,\,P) < \varepsilon \]

이 되는 \(P\) 를 잡을 수 있다.


15.2 Measure zero 와 Content zero

15.2.1 Measure zero

정의 15.6 (Measure zero) \(A\subset \mathbb{R}^n\) 을 생각하자. 만약 각각의 \(\varepsilon>0\) 에 대해 닫힌 \(n\)-cell 의 집합 \(\mathcal{U}=\{U_1,\,U_2,\ldots\}\) 가 존재하여 다음의 두 조건을 만족한다면 \(A\) 는 measure zero 라고 한다.

 (\(1\)) \(A \subset \bigcup_{U_i\in \mathcal{U}}U_i\),

 (\(2\)) \(\sum_{U_i\in \mathcal{U}} \text{vol}(U_i) < \varepsilon\)


즉 \(A\) 가 가산 무한(countably infinite) 개의 집합족 \(\mathcal{U}\) 로 덮일 수 있으며 집합족의 부피의 합이 임의로 작아질 수 있다면 \(A\) 는 measure zero 이다. Measure \(0\) 인 집합에 대해 아래 명제들은 쉽게 보일 수 있다.

명제 15.2 \(A\subset \mathbb{R}^n\) 이 measure zero 일 때 다음이 성립한다.

 (\(1\)) \(B\subset A\) 이면 \(B\) 도 measure zero 이다.

 (\(2\)) 정의 15.6 에서 \(\mathcal{U}\) 를 닫힌 \(n\)-cell 의 집합이 아닌 열린 \(n\)-cell 의 집합라고 해도 성립한다.

 (\(3\)) \(A\subset \mathbb{R}^n\) 이 유한개의 점의 집합이면 measure zero 이다.


예제 15.1 (\(1\)) 양의 정수의 집합 \(\mathbb{Z}_+\) 는 measure zero 이다. 임의의 작은 \(\varepsilon>0\) 과 \(i\in\mathbb{Z}_+\) 에 대해 \(U_i = \left[i -\dfrac{\varepsilon}{2^{i+1}},\, i+\dfrac{\varepsilon}{2^{2+i}}\right]\) 라고 하면 \(\sum_{U_i}\text{vol}(U_i) \le \varepsilon\) 이다.

(\(2\)) 유리수의 집합 \(Q\) 가 measure zero 이다. 우리는 \(\mathbb{Q}\) 와 \(\mathbb{Z}_+\) 사이에 전단사 함수 \(f:\mathbb{Q}\to \mathbb{Z}_+\) 가 존재함을 안다. (\(1\)) 에 의해 measure zero 이다.


정리 15.2 \(A=A_1 \cup A_2 \cup \cdots\) 이며 각각의 \(A_i\) 가 measure zero 이면 \(A\) 도 measure zero 이다.

(증명). 각각의 \(A_i\) 가 가산무한개의 집합족 \(\mathcal{U}_i\) 로 덮일 수 있으며 \(A\) 가 가산무한개의 집합의 합집합이므로 가산무한개의 집합족으로 덮인다. 각각의 \(A_i\) 에 대해 닫힌 \(n\)-cell 의 집합 \(\mathcal{U}_i = \{U^{i}_1,\, U^{i}_2,\ldots\}\) 이 존재하여 \(\sum_{j}\text{vol}\left(U^{i}_j\right) < \dfrac{\varepsilon}{2^{j}}\) 이 될 수 있다. 그렇다면

\[ \mathcal{U} = \bigcup \mathcal{U}_i \]

라고 하면

\[ \sum_{U\in \mathcal{U}} \text{vol}(U) \le \sum_{j=1}^\infty \dfrac{\varepsilon}{2^{j+1}} < \dfrac{\varepsilon}{2} \]

이므로 \(A\) 도 measure zero 이다. \(\square\)


15.2.2 Content zero

정의 15.7 (Content zero) \(A\subset \mathbb{R}^n\) 을 생각하자. 각각의 \(\varepsilon>0\) 에 대해 유한개의 닫힌 \(n\)-cell 의 집합 \(\mathcal{U} = \{U_1,\ldots,\,U_m\}\) 이 존재하여 다음의 두 조건을 만족하면 \(A\) 를 content zero 라고 한다.

 (\(1\)) \(A \subset \bigcup_{U_i\in \mathcal{U}}U_i\),

 (\(2\)) \(\sum_{U_i\in \mathcal{U}} \text{vol}(U_i) < \varepsilon\)


아래 명제는 쉽게 보일 수 있다.


명제 15.3 \(A\subset \mathbb{R}^n\) 이 content zero 일 때 다음이 성립한다.

 (\(1\)) \(A\) 는 measure zero 이다.

 (\(2\)) 정의 15.7 에서 \(\mathcal{U}\) 를 닫힌 \(n\)-cell 의 집합이 아닌 열린 \(n\)-cell 의 집합라고 해도 성립한다.

 (\(3\)) \(A\subset \mathbb{R}^n\) 이 유한개의 점의 집합이면 content zero 이다.


예제 15.2 \(a<b\) 에 대해 \([a,\,b]\subset \mathbb{R}\) 은 content zero 가 아니다. \(\mathcal{U}=\{U_1,\ldots,\,U_m\}\) 이 \([a,\,b]\) 에 대한 닫힌 \(1\)-cell 의 집합, 즉 닫힌 구간의 집합이라고 하자. 각각의 \(U_i\) 의 끝점을 \(a=t_0 < t_1< \cdots <t_k=b\) 가 되도록 놓을 수 있다. \(\text{vol}(U_i) \ge \min \{t_{j}-t_{j-1}:j=1,\ldots,\,m\}\) 이므로 \(\sum_{i}\text{vol} (U_i)\) 를 임의로 작게 할 수 없다. \(\square\)


명제 15.4 \(A\subset \mathbb{R}^n\) 이 measure zero 이고 옹골 집합이면 \(A\) 는 content zero 이다.

(증명). \(A\) 가 measure zero 이고 열린 \(n\)-cell 의 집합 \(\mathcal{U}=\{U_1,\ldots\}\) 가 \(A\) 의 열린 덮개이며 \(\sum_{U_i \in \mathcal{U}} \text{vol}(U_i)<\varepsilon\) 이라고 하자. \(A\) 가 옹골집합이므로 유한개의 \(\{U_1,\ldots,\,U_m\}\) 으로 덮어지며 따라서 그 부피의 합도 \(\varepsilon\) 보다 작다. \(\square\)


예제 15.3 명제 15.4 의 역은 성립하지 않는다. 즉 measure zero 이지만 content zero 가 아닌 집합이 존재한다. \(A\) 를 \([0,\,1]\) 사이의 유리수의 집합이라고 하자. \(A\subset \mathbb{Q}\) 이며 \(\mathbb{Q}\) 가 measure zero 이므로(예제 15.1) \(A\) 도 measure zero 이다. \(A\) 가 content zero 라면 유한개의 \(\{[a_1,\,b_1],\ldots,\,[a_m,\,b_m]\}\) 에 대해 \([0,\,1]\subset \bigcup_{j=1}^m [a_i,\,b_i]\) 이어야 한다. 즉 \(\sum \text{vol} \ge 1\) 이므로 \(A\) 는 content zero 가 아니다.

연습문제 15.8 (Spivak 3.8) 각각의 \(i\in \{1,\ldots,\,n\}\) 에 대해 \(a_i<b_i\) 라면 \([a_1,\,b_1]\times \cdots \times [a_n,\,b_n]\) 는 content zero 가 될 수 없음을 보여라.

(해답). \(\text{vol}([a_1,\,b_1]\times \cdots \times [a_n,\,b_n]) = (b_1-a_1)\times \cdots \times (b_n-a_n)>0\)


연습문제 15.9 (Spivak 3.9) (\(a\)) 유계가 아닌 집합은 content zero 일 수 없음을 보여라.

(\(b\)) measure zero 이지만 content zero 가 아닌 닫힌 집합의 예를 들어라.

연습문제 15.10 (Spivak 3.10) (\(a\)) \(C\) 가 content zero 집합이라면 \(C\) 의 경계도 content zero 임을 보여라.

(\(b\)) Measure zero 이며, 유계이지만 \(C\) 의 경계가 measure zero 가 아닌 집합의 예를 보여라.

(해답). (\(a\)) \(n\)-cell 의 집합 \(\mathcal{U}=\{U_1,\ldots,\,U_m\}\) 가 \(C\) 의 덮개이고 \(U=\bigcup_{i=1}^n U_i\) 는 닫힌 집합이다. \(C\subset U\) 이며 \(\overline{C}=C\cup \partial C\subset U\) (명제 6.8) 이므로 \(\partial C\subset U\) 이다. 따라서 \(C\) 의 경계도 content zero 이다.

(\(b\)) \(A=\mathbb{Q}\cap [0,\,1]\). \(\mathbb{Q}\) 가 measure zero 이므로 \(A\) 도 measure zero 이며 유계이다. 그러나 \(\partial A=[0,\,1]\) 이므로 measure zero 가 아니다.


연습문제 15.11 (Spivak 3.11) \(A\) 가 연습문제 12.12 의 \(A\) 라고 하자. \(\sum_{i=1}^\infty (b_i-a_i)<1\) 이면 \(A\) 의 경계는 measure zero 가 아님을 보여라.

(해답). \(A=\bigcup_{i=1}^\infty (a_i,\,b_i)\) 이다. 연습문제 12.12 에 의해 \(\partial A=[0,\,1]-A\) 이며 \(\text{vol}(A) <1\) 이므로 \(\text{vol}(\partial A) >0\) 이다. 따라서 \(\partial A\) 는 measure \(0\) 이 아니다.


연습문제 15.12 (Spivak 3.12) \(f:[a,\,b] \to \mathbb{R}\) 이 증가함수이다. \(\{x: f\text{ is discontinuous at }x\}\) 는 measure zero 임을 보여라.

(해답). 정해진 자연수 \(n\) 에 대해 \(X_n = \{x:o(f,\,x)> \frac{1}{n}\}\) 라고 하자. 명제 12.5 로부터 \(X_n \subset X\) 이다.

\(X_n = \{x_1,\,x_2,\ldots\}\) 이고 \(x_1<x_2<\cdots\) 라고 하면 연습문제 12.22 에 따라

\[ f(b)-f(a)>f(x_k)- f(x_1) \ge \sum_{j=2}^k f(x_j)-f(x_{j-1})> \dfrac{k-1}{n} \]

이다. \(X_n\) 이 무한집합이면 \(f\) 가 유계가 아니므로 \(X_n\) 은 유한집합이다. \(\displaystyle X=\bigcup_{i=1}^n X_n\) 이므로 정리 15.2 에 따라 \(X\) 는 measure zero 이다.

(\(b\)) 집합 \(A\subset \mathbb{R}^n\) 과 그 열린 덮개인 집합족 \(\mathcal{O}\) 를 생각하자. \(A\) 를 덮는 \(\mathcal{O}\) 의 부분집합인 수열 \(U_1,\,U_2,\ldots\) 가 존재함을 보여라.

(해답). (\(a\)) \(\mathbb{Q}^{2n}\) 은 가산집합이다.

(\(b\)) 각각의 \(x\in A\) 에 대해 \(x\) 를 포함하는 \(U_x\in \mathcal{O}\) 가 존재하며 \(x\in B_x \subset U_x\) 인 경계가 유리수인((\(a\)) 에서 말한 그것) \(n\)-cell \(B_x\) 가 존재한다. 이런 \(B_x\) 에 대해 \(U_x\) 를 하나씩 모으면 \(B_x\) 의 집합은 countable 이므로 이런 \(U_x\) 의 집합도 countable 이다.


15.3 적분가능한 함수

15.3.1 적분가능한 함수


보조정리 15.1 닫힌 \(n\)-cell \(A\subset \mathbb{R}^n\) 에서 정의된 함수 \(f:A \to \mathbb{R}\) 이 유계이며 모든 \(\boldsymbol{x}\in A\) 에서 \(o(f,\,\boldsymbol{x}) <\varepsilon\) 이면 어떤 \(P\in \mathcal{P}[A]\) 가 존재하여 \(U(f,\,P) - L(f,\,P)<\varepsilon \cdot \text{vol}(A)\) 를 만족한다.

(증명). \(\boldsymbol{x}\in A\) 이면 \(\boldsymbol{x}\) 를 내부에 포함하는 어떤 닫힌 \(n\)-cell \(U_x\) 에서 \(M_{U_x}(f) - m_{U_x}(f) <\varepsilon\) 이다. \(A\) 가 옹골집합이므로 유한개의 \(U_{x_1},\ldots,\,U_{x_n}\) 이 존재하여 \(A\) 를 덮는다. \(A\) 의 분할 \(P\) 를 \(P\) 의 각각의 subcell 이 어떤 \(U_{x_i}\) 에 포함되도록 잡을 수 있다. \(S\) 가 그런 subcell 이라면 \(M_S(f)-m_S(f)<\varepsilon\) 이며, 따라서

\[ U(f,\,P)-L(f,\,P) = \sum_{S} \left[M_S(f)-m_S(f)\right] \cdot \text{vol}(S) < \varepsilon \cdot \text{vol}(A) \]

이다. \(\square\)


정리 15.3 닫힌 \(n\)-cell \(A\subset \mathbb{R}^n\) 에서 정의된 유계인 함수 \(f:A \to \mathbb{R}\) 을 생각하자. \(B=\{\boldsymbol{x}\in A:f\text{ is not continuous at }f\}\) 일 때 \(f\) 가 적분가능한 것의 필요충분조건은 \(B\) 가 measure zero 인 것이다.

(증명). \(B_\varepsilon := \{\boldsymbol{x}\in A: o(f,\,\boldsymbol{x})\ge \varepsilon\}\) 라고 하자.

(\(1\)) 우선 \(B\) 가 measure zero 임을 가정하자 \(\varepsilon>0\) 에 대해 \(B_\varepsilon \subset B\) 이므로 \(B_\varepsilon\) 은 measure zero 이다. 명제 12.6 에 의해 \(B_\varepsilon\) 은 닫힌집합이며 \(B_\varepsilon \subset A\) 이므로 유계이다. 따라서 \(B_\varepsilon\) 은 옹골집합이므로 content zero 이다(명제 15.4). 따라서 유한개의 닫힌 \(n\)-cell \(U_1,\ldots,\,U_n\) 로 \(\sum_{i=1}^n \text{vol}(U_i)<\varepsilon\) 이며 \(B_\varepsilon\) 을 덮을 수 있다. 이제 \(A\) 에 대한 분할 \(P\) 를 \(P\) 에 따라 정해지는 subcell 이 아래의 \(\mathcal{S}_1\) 이나 \(\mathcal{S}_2\) 중의 하나에 속하도록 잡자.(그림 15.1)

(\(1\)) \(\mathcal{S}_1\) : 어떤 \(U_i\) 에 대해 \(S \subset U_i\) 이면 \(S\) 는 \(\mathcal{S}_1\) 에 포힘된다.

(\(2\)) \(\mathcal{S}_2\) : \(S\cap B_\varepsilon = \varnothing\) 이면 \(S\) 는 \(\mathcal{S}_2\) 에 포함된다.

이제 \(|f(\boldsymbol{x})|<M\) 이라고 하자. 각각의 subcell \(S\) 에 대해 \(M_S(f)-m_S(f)<2M\) 이므로

\[ \sum_{S\in \mathcal{S}_1} \left[M_S(f) - m_S(f) \right]\cdot \text{vol}(S) < 2M \sum_{i=1}^n \text{vol}(U_i) < 2M\varepsilon \]

이제 \(S\in \mathcal{S}_2\) 이면 \(o(f,\boldsymbol{x})<\varepsilon\) 이므로 보조정리 15.1 에 따라 \(P\) 의 어떤 refinement \(P'\) 이 존재하여 \(S\subset \mathcal{S}_2\) 일 때

\[ \sum_{S'}\subset S [M_{S'}(f) - m_{S'}(f)]\cdot \text{vol}(S') < \varepsilon \cdot \text{vol}(S) \]

이다. 그렇다면

\[ \begin{aligned} U(f,\,P') - L(f,\,P') &= \sum_{S'\subset S \in \mathcal{S}_1} \left[M_{S'}(f) - m_{S'}(f)\right]\cdot \text{vol}(S') \\[0.3em] &\qquad + \sum_{S'\in S\in \mathcal{S}_2} \left[M_{S'}(f) - m_{S'}(f)\right]\cdot \text{vol}(S') \\[0.3em] &< 2M\varepsilon + \sum_{S \in \mathcal{S}_2} \varepsilon \cdot \text{vol}(S) \\[0.3em] &\le 2M\varepsilon + \varepsilon \cdot \text{vol}(A) \end{aligned} \]

이 성립한다. \(M\) 과 \(\text{vol}(A)\) 는 정해져 있으므로 우리는 \(U(f,\,P')-L(f,\,P')\) 을 원하는 만큼 작게 만들 수 있다. 따라서 \(f\) 는 적분가능하다.

이제 \(f\) 가 적분가능함을 가정하자. \(B= B_1 \cup B_{1/2} \cup \cdots\) 이므로 각각의 \(B_{1/n}\) 이 meausure zero 라면 정리 15.2 에 의해 \(B\) 가 measure zero 임을 알게 된다. 실제로 우리는 각각의 \(B_{1/n}\) 이 content zero 임을 보일 것이다.

\(f\) 가 적분가능하므로 \(\varepsilon>0\) 에 대해 \(P \in \mathcal{P}[A]\) 를 \(U(f,\,P)-L(f,\,P)< \varepsilon/n\) 이 되도록 잡을 수 있다. \(\mathcal{S}\) 를 \(B_{1/n}\) 과의 교집합이 공집합이 아닌 \(P\) 의 subcell 의 집합족이라고 하자. 그렇다면 \(\mathcal{S}\) 는 \(B_{1/n}\) 의 덮개이며 \(S\in \mathcal{S}\) 에 대해 \(M_S(f)-m_S(f)\ge 1/n\) 이다. 그렇다면

\[ \begin{aligned} \dfrac{1}{n} \cdot \sum_{S\in \mathcal{S}} \text{vol}(S) & \le \sum_{S\in \mathcal{S}} \left[M_S(f) - m_S(f)\right] \cdot \text{vol}(S) \\[0.3em] &\le \sum_{S} \left[M_S(f)- m_S(f)\right]]\cdot \text{vol}(S) < \dfrac{\varepsilon}{n} \end{aligned} \]

이므로 결론적으로 \(\sum_{S\in \mathcal{S}} \text{vol}(S)< \varepsilon\) 이다. \(\square\)


15.3.2 특성함수와 적분 가능성

지금까지는 \(n\)-cell 위에서의 적분만을 생각했다. 이제 \(n\)-cell 에서의 적분을 확장하여 임의의 집합에 대한 적분에 대해 알아보자.

정의 15.8 (특성함수와 집합의 적분) 아래와 같이 정의된 함수 \(\chi_A:A\subset \mathbb{R}^n \to \mathbb{R}\) 를 \(A\) 에서의 특성 함수 (characteristic fucntion) 라고 한다.

\[ \chi_A (x) := \left\{\begin{array}{ll} 0, \qquad & x\in X,\\ 1 & x\in C\end{array}\right. . \tag{15.5}\]

어떤 닫힌 직육면체 \(C\) 에 대해 \(A\subset C\) 이고 \(f:C\to \mathbb{R}\) 이 유계일 때 \(f\cdot \chi_A\) 가 적분 가능하다면 \(A\) 에서의 \(f\) 의 적분은 아래와 같이 정의된다.

\[ \int_A f := \int_C f \cdot \chi_A \tag{15.6}\]


명제 15.5 \(A\) 에서 정의된 특성함수 \(\chi_A\) 가 적분 가능할 필요충분조건은 \(\partial A\) 가 measure \(0\) 인 것이다.

(증명). \(x\in \text{int}(A)\) 이거나 \(x\) 가 \(A\) 의 외부라면 \(\chi_A\) 가 \(x\) 에서 연속임은 자명하다. \(x\in \partial A\) 라면 \(\chi_A\) 는 \(x\) 에서 연속이지 않으며 정리 15.3 에 따라 명제가 증명된다. \(\square\)


정의 15.9 (조르당 가측) 집합 \(A\) 가 유계이고 \(\partial A\) 가 measure \(0\) 일 때 \(A\) 를 조르당 가측(Jordan-measureable) 이라고 한다. \(\int_A 1\) 을 \(A\) 의 content 혹은 \(n\)-차원 부피(volumn) 라고 한다. \(1\) 차원일 경우는 길이(length), 2차원일 경우는 넓이(area) 라고 한다.

15.4 푸비니의 정리

15.4.1 직사각형 영역에서 정의된 함수의 적분

직사각형 \([a,\,b] \times [c,\,d]\) 위에서 정의된 실함수 \(f:[a,\,b] \times [c,\,d] \to \mathbb{R}\) 을 생각하자. \((t_0,\ldots,\,t_n)\) 이 \([a,\,b]\) 의 분할이라고 할 때 \(\{t_i\} \times [c,\,d]\) 는 \([a,\,b]\times [c,\,d]\) 를 \(n\) 개의 긴 직사각형 조각으로 나눈다. 정해진 \(x\in [a,\,b]\) 에 대해 \(g_x(y)=f(x,\,y)\) 라고 하자. \([x] \times [c,\,d]\) 위에서 \(f\) 그래프까지의 면적은 아래와 같다.

\[ \int_c^d g_x = \int_c^d f(x,\,y)\, dy \]

또한 \([t_{i-1},\,t_i]\times [c,\,d]\) 영역 위에서 \(f\) 아래 영역의 부피는 \(x\in [t_{i-1},\,t_i]\) 에 대해 대략

\[ (t_{i}-t_{i-1}) \cdot \int_c^d f(x,\,y) \,dy \]

이며, 따라서

\[ \int_{[a,b]\times [c,d]} f = \sum_{i=1}^n \int_{[t_{i-1},t_i]\times [c,,d]} f \approx \sum_{i=1}^n (t_{i-1},t_i) \cdot \int_c^d f(x,\,y)\, dy \]

임을 안다. 다른 한편 위의 식의 \(\sum_i\) 관련 부분은 \(\int_a^b \left(\int_c^d f(x,\,y)\,dy\right)\,dx\) 의 정의에서 나타나는 그것과 유사하다. 따라서 \(h(x) := \int_c^d g_x = \int_c^d f(x,\,y)\,dy\) 라고 정의하면 \(h \in \mathcal{R}[a,\,b]\) 이고

\[ \int_{[a,b]\times [c,d]} f= \int_a^b h = \int_a^b \left(\int_c^d f(x,\,y)\, dy\right)\,dx \]

임을 기대할 수 있다. 우리가 앞으로 할 일은 \(f\) 가 연속일 경우 이것이 사실임을 보이는 것인데, 일반적인 경우에서는 그렇지 않을 수 있다. \(f:[a,\,b] \times [c,\,d] \to \mathbb{R}\) 이 불연속인 점이 \([x]\times [c,\,d]\) 라고 하자. 그랗다면 \(f\) 는 \([a,\,b]\times [c,\,d]\) 에서는 적분 가능하지만 \(h(x_0) = \int_c^d f(x_0,\,y)\,dy\) 는 정의 자체가 안될 수도 있다. 앞으로 보겠지만 푸비니의 정리 자체는 이것을 생각하면 약간 의아할 수 있고, 여러 특별한 경우에 대한 주의사항이 붙게 된다.

푸비니의 정리를 진행하기에 앞서 리만 상합과 리만 하합(정의 15.4) 을 다시 한번


정리 15.4 (푸비니 정리) \(A\subset \mathbb{R}^m,\,B\subset \mathbb{R}^n\) 이 닫힌 \(n\)-cell 이고 \(f:A \times B \to \mathbb{R}\) 가 적분가능하다고 하자. \(\boldsymbol{x}\in A\) 에 대해 \(g_\boldsymbol{x}:B \to \mathbb{R}\) 을 \(g_\boldsymbol{x}(\boldsymbol{y}):=f(\boldsymbol{x},\boldsymbol{y})\) 로 정의하자. 그리고

\[ \begin{aligned} \mathcal{L}(\boldsymbol{x}) &= \underline{\int_B}g_\boldsymbol{x},\\[0.3em] \mathcal{U}(\boldsymbol{x}) & = \overline{\int_B}g_\boldsymbol{x} \end{aligned} \]

라고 하면 \(\mathcal{L}\) 과 \(\mathcal{U}\) 는 \(A\) 에서 적분가능하며

\[ \begin{aligned} \int_{A\times B} f = \int_A \mathcal{L}(\boldsymbol{x}) \,d\boldsymbol{x} = \int_A \mathcal{U}(\boldsymbol{x}) \,d\boldsymbol{x} \end{aligned} \]

이다.

(증명). \(P_A \in \mathcal{P}(A),\, P_B \in \mathcal{P}(B)\) 라고 하자. \(P_A\times P_B \in \mathcal{P}(A \times B)\) 이다. \(S_A,\, S_B\) 를 각각 \(P_A\), \(P_B\) 의 subcell 이라고 하면 \(P_A \times P_B\) 에 속하는 subcell \(S\) 는 \(S_A\times S_B\) 꼴이다. 따라서

\[ \begin{aligned} L(f,\,P) &= \sum_S m_S(f)\cdot \text{vol}(S) = \sum_{S_A,S_B} m_{S_A\times S_B}(f) \cdot \text{vol}(S_A \times S_B) \\[0.3em] &= \sum_{S_A}\left( \sum_{S_B} m_{S_A \times S_B}(f) \cdot \text{vol}(S_B)\right) \cdot \text{vol}(S_A) \end{aligned} \]

이다. \(\boldsymbol{x}\in S_A\) 이면 \(m_{S_A\times S_B}(f) \le m_{S_B}(g_\boldsymbol{x})\) 이므로

\[ \sum_{S_B} m_{S_A\times S_B}(f)\cdot \text{vol}(S_B) \le \sum_{S_B} m_{S_B}(g_\boldsymbol{x})\cdot \text{vol}(S_B) \le \mathcal{L}(\boldsymbol{x}) \]

이다. 즉

\[ \sum_{S_A}\left(\sum_{S_B} m_{S_A\times S_B}(f)\cdot \text{vol}(S_B)\right) \cdot \text{vol}(S_A) \le L(\mathcal{L},\, P_A) \]

같은 방법으로 \(U(\mathcal{U},\,P_A)\) 에 대해 구하면 다음을 얻는다.

\[ L(f,\,P) \le L(\mathcal{L},P_A) \le U(\mathcal{L}, P_A) \le U(\mathcal{U},P_A) \le U(f,\,P). \]

\(f\) 가 적분가능하므로 \(\sup L(f,\,P) = \inf U(f,\,P) = \int_{A\times B}f\) 이며, 따라서

\[ \sup L(\mathcal{L},\,P_A) = \inf U(\mathcal{L},\, P_A) \]

이다. \(\square\)


아래 따름 정리는 푸비니 정리로부터 쉽게 보일 수 있다.

따름정리 15.1 푸비니 정리에서 \(x\) 와 \(y\) 변수 사이에 구별이 없다. 즉 \(h_\boldsymbol{y}(\boldsymbol{x}) := f(\boldsymbol{x},\boldsymbol{y})\) 라고 할 때

\[ \begin{aligned} \int_{A\times B} f &= \int_A L(g_\boldsymbol{x},\,B)\,d\boldsymbol{x} = \int_{A}U(g_\boldsymbol{x},\,B)\,\boldsymbol{x}\\[0.3em] &= \int_B L(h_\boldsymbol{y},\,A)\,d\boldsymbol{y} = \int_B U(h_\boldsymbol{y},\,A)\,d\boldsymbol{y} \end{aligned} \tag{15.7}\]

가 성립함을 안다. 즉 \(f\) 가 적분가능 함수라면 적분의 순서를 바꿀 수 있다.


주석 15.1 (푸비니 정리에 대하여).
(\(1\)) 각각의 \(g_\boldsymbol{x}\) 가 적분가능하다면

\[ \int_{A\times B} f = \int_A \left(\int_B f(\boldsymbol{x},\boldsymbol{y})\,d\boldsymbol{y}\right)\,d\boldsymbol{x} \]

이다. \(f\) 가 연속일경우는 당연히 \(g_\boldsymbol{x}\) 는 적분가능하다.

(\(2\)) 최악의 경우는 유한개의 \(\boldsymbol{x}\in A\) 에 대해 \(g_\boldsymbol{x}\) 가 적분가능하지 않은 경우이다. 이 경우 \(\mathcal{L}(\boldsymbol{x})=\int_B f(\boldsymbol{x},\boldsymbol{y})\,d\boldsymbol{y}\) 는 유한개의 \(\boldsymbol{x}\) 이외에서 존재한다. 이 \(\boldsymbol{x}\) 에 대해 \(\mathcal{L}(\boldsymbol{x})=0\) 이라고 재정의 하면 \(\int_A \mathcal{L}\) 은 변화가 없기 때문이 \(\int_{A\times B}f = \int_A \left(\int_B f(\boldsymbol{x},\boldsymbol{y})\,d\boldsymbol{y}\right)\,d\boldsymbol{x}\) 라고 쓸 수 있다.

(\(3\)) 푸비니 정리(정리 15.4) 가 작동하지 않는 경우를 살펴보자. 우선 \(f:[0,\,1]\times [0,\,1] \to \mathbb{R}\) 이 아래와 같이 정의되었다고 하자.

\[ f(x,\,y) = \left\{\begin{array}{ll} 1, &x\in \mathbb{R}\backslash\mathbb{Q}\\ 1, & x\in \mathbb{Q},\, y \in \mathbb{R}\backslash \mathbb{Q}\\ 1-\dfrac{1}{q},\qquad &y\in \mathbb{Q},\, x=\dfrac{p}{q}\end{array}\right. \]

여기서 마지막의 \(x=p/q\) 는 기약 상태일 때이다. 만약 \(f\) 가 적분가능하다면 \(\int_{[0,\,1]\times [0,1]}f=1\) 이다. \(x \in \mathbb{R}/\backslash Q\) 라면 \(\int_0^1 f(x,\,y)dy = 1\) 이며 \(x\in \mathbb{Q}\) 이면 \(\int_0^1 f(x,\,y)dy = 0\) 이다. \(h(x) :=\int_0^1 f(x,\,y)\,dy\) 에 대해 이 적분이 존재하지 않을 때 \(h(x)=0\) 으로 놓는다면 \(h\) 는 적분가능하지 않다.

(\(4\)) \(A=[a_1,b_1] \times \cdots \times [a_n,b_n]\) 에 대해 \(f:A\to \mathbb{R}\) 이 충분히 좋다면 푸비니의 정리를 반복적으로 사용하여 다음을 얻는다.

\[ \int_A f = \int_{a_n}^{b_n} \left(\cdots \left(\int_{a_1}^{b_1} f(x_1,\ldots,\,x_n)\, dx_1\right)\cdots\right)\,dx_n. \tag{15.8}\]

(\(5\)) \(C \subset A \times B\) 일 때 \(\int_C f = \int_A f \cdot \chi_C\) 이므로 푸미니의 정리를 이용하여 \(\int_C f\) 를 계산 할 수 있다. 예를 들어

\[ \begin{aligned} A=B=[0,\,1],\qquad C=\{(x,\,y)\in A\times B : \|(x,\,y)\|<1\} \end{aligned} \]

라면

\[ \int_C f = \int_{-1}^1 \left(\int_{-1}^1 f(x,\,y) \cdot \chi_C(x,\,y) \,dy\right)\,dx \]

이다. 이 때

\[ \chi_C (x,\,y) = \left\{\begin{array}{ll} 1,\qquad& \text{if } y > \sqrt{1-x^2} \text{ or } y<-\sqrt{1-x^2}, \\ 0, & \text{otherwise}\end{array} \right. \]

라고 놓을 수 있으며, 따라서

\[ \int_{-1}^1 f(x,\,y)\cdot \chi_C(x,\,y)\, dy = \int_{-1}^{-\sqrt{1-x^2}} f(x,\,y)\,dy + \int_{\sqrt{1-x^2}}^1 f(x,\,y)\, dy \]

이다. 일반적으로 \(C\subset A \times B\) 에 대해 \(\int_C f\) 를 계산하는데의 어려운 점은 \(x\in A\) 에 대해 \(C\cap (\{x\}\times B)\) 를 결정하는 것이다. 만약 \(y\in B\) 에 대해 \(C \cap (A\times \{y\})\) 의 경우가 더 쉽다면

\[ \int_C f = \int_B \left(\int_A f(x,\,y) \cdot \chi_C (x,\,y)\,dx\right)\,dy \]

를 사용하는 것이 좋다.
